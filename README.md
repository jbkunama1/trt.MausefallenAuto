# Mausefallenauto Lern-App

Moderne, schulgeeignete Web-App zum Thema **Mausefallenauto / Mousetrap Car** für den Technikunterricht in der Sekundarstufe I.

## Inhalt

- interaktive HTML-Webseite für den Unterricht
- didaktisch-methodische Aufbereitung
- Bauanleitung in Unterrichtssprache
- Hinweise zu 3D-druckbaren Thingiverse-Modellen
- NotebookLM-Dokument für weiterführende Medienarbeit

## Dateien

- `mausefallenauto-lernapp.html` – zentrale Web-App
- `notebooklm-prompt-und-kontext.md` – Prompt und Projektdokument für NotebookLM
- `quellen.md` – kompakte Dokumentation der verwendeten Onlinequellen

## Einsatz in der Schule

Geeignet für Technikunterricht in Klasse 7–9, besonders im Bereich Mobilität, Antrieb, Konstruktion, Erprobung und Optimierung.

## Veröffentlichung auf GitHub

1. Dateien in ein neues Repository legen
2. Optional Bilder der Thingiverse-Modelle im Ordner `assets/` ergänzen
3. README bei Bedarf um eigene Screenshots, Unterrichtsmaterialien und Lizenzhinweise erweitern
